App gratuito para a criação de fóruns similar ao Reddit

É uma rede encriptada P2P com moderação e base de dados distribuída.

15.03.16 | João dos Santos

Aether é um app gratuito para a criação de fóruns similar ao Reddit, mas que não possui servidores. É uma rede encriptada P2P com moderação e base de dados distribuída. Os usuários permanecem anônimos, e todos os conteúdos publicados são automaticamente licenciados em CC, sendo deletados após seis meses.
